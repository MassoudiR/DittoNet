DittoNet V1

The Ultimate Interception & Mocking Gateway

🚀 1. Introduction & Setup

DittoNet V1 acts as a localized API gateway for your HTTP clients (such as Flutter applications, mobile apps, or web frontends). By routing your app's traffic through DittoNet instead of hitting external APIs directly, you gain the superpower to instantly inspect, modify, mock, or block requests and responses in real-time.

Installation

pip install flask flask-socketio requests
python -m ditto_interceptor

By default, the server runs on http://0.0.0.0:5000. Open this URL in your browser to view the Dashboard.

🖥️ 2. The Dashboard

The web dashboard provides a live look at all traffic flowing through the server.

📡 Live Traffic Feed

Requests appear instantly. Click on any log to expand it. You can inspect exact JSON payloads, headers, and status codes. If a request was modified by a rule, you will see a MODIFIED badge.

🔄 Replay Engine

Every intercepted log features a Replay button. This opens a modal allowing you to tweak the method, headers, URL, or body and immediately resend the request to test endpoint behavior without having to restart your mobile app.

🛡️ 3. Rules Engine

Rules allow you to define UI-based logic that triggers on specific endpoints. You can use wildcard matching like */api/v1/auth/*.

✅ ALLOW

Explicitly allows the traffic to pass untouched. Useful to override broader blocking rules.

⛔ BLOCK

Instantly aborts the request. Your app will receive an empty response with a 403 or similar status code, ensuring the remote server is never reached.

🔀 REDIRECT

Intercepts a call to a production URL and reroutes it entirely (e.g. redirecting https://api.prod.com to http://localhost:8080).

📝 MATCH_REPLACE

Finds a string anywhere in the body (JSON, HTML, JS) and replaces it. Perfect for flipping flags like "isHacked": false to "isHacked": true.

📦 BODY_REPLACE

Completely overwrites the payload body. The dashboard provides a "Load Req/Res Body" button to easily copy the original body into the editor.

💉 HEADER_INJECT

Injects custom HTTP headers. Format: Key: Value.

🕵️ 4. Programmatic Inspectors

Sometimes UI rules aren't enough, and you want to hardcode interception logic directly in Python. Inspectors are functions triggered automatically by URL patterns.

Use the @server.inspector decorator in your server initialization. Inspectors run before database rules.

# Define an inspector that triggers on any request containing '/api/play/'
@server.inspector(pattern="*/api/play/*", method="ANY", phase="Both")
def inspect_gameplay(flow_id, phase, headers, body):
    print(f"Inspecting Gameplay | Phase: {phase}")
    
    # Example: Auto-modify a response payload
    if phase == "Response" and isinstance(body, dict):
        if "coins" in body:
            body["coins"] = 999999 # Infinite coins cheat!
            
    # Must return a tuple of (headers, body)
    return headers, body

🧩 5. Custom Plugins

Unlike Inspectors (which trigger automatically), Plugins are assigned to specific URLs manually via the UI Rules Engine.

@server.plugin("DecryptPayload")
def decrypt_plugin(flow_id, phase, headers, body):
    # Implement custom crypto logic here
    if isinstance(body, dict) and "encrypted_data" in body:
        body["decrypted_data"] = "hello world"
        
    # Plugins must return a tuple: (new_headers, new_body)
    return headers, body

Once defined in your python code, DecryptPayload will be selectable from the "EXECUTE PLUGIN" dropdown in the Dashboard Rule Editor.

📱 6. Flutter Integration Example

To make this magic happen, your Flutter app must use an HTTP Interceptor (like Dio) to pass traffic through DittoNet.

// Example using the 'dio' package in Flutter
import 'package:dio/dio.dart';
import 'package:uuid/uuid.dart';

class DittoInterceptor extends Interceptor {
  final String dittoServerUrl = 'http://192.168.1.100:5000';
  final _dio = Dio();
  final _uuid = Uuid();

  @override
  void onRequest(RequestOptions options, RequestInterceptorHandler handler) async {
    final flowId = _uuid.v4();
    options.extra['flowId'] = flowId; // Save to link with response phase

    // 1. Ask DittoNet to process the Request
    try {
      final result = await _dio.post('$dittoServerUrl/api/intercept/request', data: {
        'flowId': flowId,
        'url': options.uri.toString(),
        'method': options.method,
        'headers': options.headers,
        'body': options.data,
      });

      final dittoData = result.data;
      
      // Abort request if DittoNet blocked it
      if (dittoData['action'] == 'BLOCK') {
        return handler.reject(DioException(
            requestOptions: options, 
            error: 'Blocked by DittoNet'
        ));
      }

      // Handle URL Redirect
      if (dittoData['action'] == 'REDIRECT' && dittoData['redirectUrl'] != null) {
        options.path = dittoData['redirectUrl'];
      }

      // Apply modifications
      options.headers = Map.from(dittoData['headers']);
      options.data = dittoData['body'];
      
      return handler.next(options);
    } catch (e) {
      return handler.next(options); // Fallback to normal if DittoNet is down
    }
  }

  @override
  void onResponse(Response response, ResponseInterceptorHandler handler) async {
    final flowId = response.requestOptions.extra['flowId'];
    
    // 2. Ask DittoNet to process the Response
    try {
      final result = await _dio.post('$dittoServerUrl/api/intercept/response', data: {
        'flowId': flowId,
        'url': response.requestOptions.uri.toString(),
        'method': response.requestOptions.method,
        'statusCode': response.statusCode,
        'headers': response.headers.map,
        'body': response.data,
      });

      final dittoData = result.data;
      
      // Apply spoofed data to response before app reads it
      response.headers = Headers.fromMap(
          Map>.from(dittoData['headers'])
      );
      response.data = dittoData['body'];
      
      return handler.next(response);
    } catch (e) {
      return handler.next(response);
    }
  }
}

Simply add this interceptor to your Dio instance: dio.interceptors.add(DittoInterceptor());
