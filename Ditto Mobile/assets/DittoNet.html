01. Core Engine Architecture

02. Optimizing Functions & Workflow
A

Multi-Tab Memory Optimization

DittoNet utilizes an intelligent IndexedStack preservation controller. Background tabs retain their scroll offset, form data, and DOM state without draining CPU cycles. Tap the tab counter pill in the top navbar to switch or spawn new independent sessions.

B

Glassmorphic Link Interception & Text Selection

Long-press any anchor tag or URL string across web pages to invoke the custom frosted glass modal sheet. Effortlessly copy clean link URLs or launch them directly into background tabs while preserving native text selection for standard body content.

C

Per-Origin Granular Permissions

Tap the SSL Lock icon inside the expandable URL bar to configure site-specific security policies. Toggle Camera, Microphone, Geolocation, and JavaScript permissions individually per hostname to prevent unwanted fingerprinting.

D

HAR 1.2 Traffic Exporting

Capture full request and response payloads, status codes, timing metrics, and headers. Open the Network Suite menu to export complete sessions as standardized .har files compatible with Wireshark and Google Chrome DevTools.
